About Tej Kumar

Tej Kumar writes intimate tales exploring memory, love and consequence. This site shares stories and books freely — readers only need an account to access full PDFs.

Philosophy

Stories are meant to be shared. This collection focuses on human-scale narratives, character-driven tales, and short-form experiments.

How this site works

Books are linked as PDFs (Drive or repo). Admin can add new books via the Upload page. The site uses localStorage for metadata in this demo; for production we recommend Supabase or Firebase for storage and auth.
