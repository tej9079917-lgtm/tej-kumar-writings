# Tej Kumar Writings — Static Website

This package contains a ready-to-deploy static website for GitHub Pages.

Files:
- index.html
- library.html
- story.html
- upload.html
- admin.html
- about.html
- contact.html
- login.html
- signup.html
- style.css
- script.js
- assets/Mr-Romeo.pdf (sample if available)
- assets/mr-romeo-cover.png (sample if available)

Usage:
1. Upload this folder to your GitHub repository root.
2. Enable GitHub Pages on the repository (main branch, root).
3. Visit the site. Use Upload page to add books via Google Drive links.

Notes:
- This demo stores metadata in localStorage. For production use a backend (Supabase/Firebase) for storage and authentication.
